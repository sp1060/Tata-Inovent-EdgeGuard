# Battery Management System (BMS) - Edge Model
This module handles feature engineering and battery health (SoH) prediction.

## 📊 Dataset Reference
The model is trained on public datasets. Download the source files here:
* **NASA Battery Aging Dataset:** [NASA Repository Link](https://www.nasa.gov/intelligent-systems-division/discovery-and-systems-health/pcoe/pcoe-data-set-repository/)
* **CALCE Battery Data:** [UMD CALCE Link](https://calce.umd.edu/battery-data)
